# Mail-Nachfolge-Wizard

Browser-Tool zum Erstellen von Support-E-Mails für das Onboarding von Booking-Engine-Kunden.

## Nutzung

`index.html` im Browser öffnen, relevante Bausteine auswählen, fertigen Text kopieren und in Pipedrive einfügen.

## Bausteine

| Kategorie | Baustein |
|---|---|
| Kommunikation & E-Mail | SMTP, E-Mail-Layout |
| Booking Engine | Allg. Einstellungen, BE Pro, Bilder, Buchungsbeschränkungen, Saisons, Dynamische Raten |
| Abrechnung & Rechtliches | Rechnungseinstellungen, Stornobedingungen, MwSt.-Sätze |

## Bilder

Die Screenshots in `assets/` sind die Quell-Dateien für Dropbox. Die Links im HTML zeigen auf Dropbox, damit Kunden die Bilder in Pipedrive-Mails sehen können.
